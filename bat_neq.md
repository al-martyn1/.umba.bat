В bat-файлах Windows для проверки неравенства переменных используется оператор `NEQ`. Вот основные способы:

## 1. Для строковых переменных

```batch
@echo off
set VAR=test

if not "%VAR%"=="value" (
    echo Переменная НЕ равна "value"
)

REM Или с использованием NEQ
if "%VAR%" NEQ "value" (
    echo Переменная НЕ равна "value"
)
```

## 2. Для числовых переменных

```batch
@echo off
set /A NUM=10

if %NUM% NEQ 5 (
    echo Число НЕ равно 5
)
```

## 3. Проверка на пустоту

```batch
@echo off
set VAR=

if "%VAR%"=="" (
    echo Переменная пустая
)

if not "%VAR%"=="" (
    echo Переменная НЕ пустая
)
```

## 4. Комплексный пример с пользовательским вводом

```batch
@echo off
set /p INPUT="Введите значение: "

if "%INPUT%" NEQ "yes" (
    echo Вы ввели НЕ 'yes'
) else (
    echo Вы ввели 'yes'
)
```

## 5. Сравнение двух переменных

```batch
@echo off
set VAR1=hello
set VAR2=world

if "%VAR1%" NEQ "%VAR2%" (
    echo Переменные разные
)
```

## Важные моменты:

- **NEQ** работает и для строк, и для чисел
- **not** инвертирует результат сравнения
- Всегда заключайте переменные в кавычки, особенно если они могут содержать пробелы
- Для числовых сравнений можно также использовать: `EQU`, `NEQ`, `LSS`, `LEQ`, `GTR`, `GEQ`

## Пример с выбором действия:

```batch
@echo off
set /p CHOICE="Выберите действие (1/2): "

if "%CHOICE%" NEQ "1" if "%CHOICE%" NEQ "2" (
    echo Неверный выбор!
    goto :end
)

if "%CHOICE%" EQU "1" (
    echo Вы выбрали вариант 1
)

if "%CHOICE%" EQU "2" (
    echo Вы выбрали вариант 2
)

:end
pause
```

Наиболее часто используется конструкция `if "%var%" NEQ "value"` для проверки неравенства строковых значений.
